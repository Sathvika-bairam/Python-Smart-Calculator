# Python-Smart-Calculator

# Smart Calculator Using Python
A simple and user-friendly Smart Calculator developed using Python. This project performs basic arithmetic operations and stores calculation history for future reference.
# Project Objective
The main objective of this project is to develop a smart calculator application using Python that can perform mathematical operations efficiently while storing user calculation history.
# Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for invalid inputs
- Saves calculation history
- User-friendly menu-driven interface
# Technologies Used
- Python
# Concepts Used
- Functions
- Loops
- Conditional Statements
- Exception Handling
- File Handling
